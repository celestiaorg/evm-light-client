# evm-light-client

![Truffle Test CI](https://github.com/lazyledger/evm-light-client/workflows/Truffle%20Test%20CI/badge.svg)

EVM-compatible on-chain light client for LazyLedger consensus

## Building

Install dependencies:

```sh
npm install
```

Build:

```sh
truffle compile
```

Test:

```sh
truffle test
```
